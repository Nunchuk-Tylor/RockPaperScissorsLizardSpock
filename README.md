# Rock–Paper–Scissors–Lizard–Spock

## Description
This is a Rock/Paper/Scissors style game where you have five options to choose from built in Java code.

## How to Run
1. Start the program
2. Check Acheivements
3. Enter one of these:
   - rock
   - paper
   - scissors
   - lizard
   - spock
3. Look at results
4. Decide to play again or not

## Game Rules
- Scissors beats Paper
- Paper beats Rock
- Rock beats Lizard
- Lizard beats Spock
- Spock beats Scissors
- Scissors beats Lizard
- Lizard beats Paper
- Paper beats Spock
- Spock beats Rock
- Rock beats Scissors
- and two of the same type tie
